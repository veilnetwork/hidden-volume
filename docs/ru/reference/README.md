# Reference — формальные спецификации

[🇬🇧 English](../../en/reference/README.md) · [🇷🇺 Русский](README.md)

Формальный справочный материал. Читайте после [`../guide/`](../guide/),
когда нужны байтовые детали или гарантии стабильности.

## Документы

- **[format.md](format.md)** — каноническая байтовая
  спецификация on-wire формата v1. Заголовки, layout chunk'а,
  AEAD-обрамление, состав AAD. Единый источник истины по байтам
  на диске.
- **[api-surface.txt](../../en/reference/api-surface.txt)** —
  frozen-снимок каждого `pub` элемента на v1.0. Используется CI
  для обнаружения случайного дрейфа API. Файл нейтрален по языку
  (Rust-идентификаторы не переводятся), поэтому хранится только в
  `docs/en/reference/`.
- **[semver.md](semver.md)** — политика semver. Что является
  ломающим изменением в линии v1.x: формат, публичный Rust API,
  cargo features, FFI ABI, варианты ошибок.
- **[ffi.md](ffi.md)** — архитектура `hidden-volume-ffi`.
  Стратегия uniffi 0.31 биндингов, маппинг ошибок, управление
  временами жизни.

## Когда что читать

| Вы хотите … | Читайте |
|---|---|
| Реализовать новый reader/writer формата | `format.md` |
| Проверить, ломает ли рефакторинг публичный API | `api-surface.txt` |
| Понять правила бамп-апа версии | `semver.md` |
| Добавить биндинг под чужой язык | `ffi.md` |
